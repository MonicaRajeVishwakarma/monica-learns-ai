# Day 1 Notes – Python Basics

## Variables
```python
name = "Monica"
baby_age_weeks = 18
milk_quantity = 0.12
```

## Data Types
```python
int → whole numbers  
float → decimal numbers  
str → text  
bool → True / False  
```

## Examples
```python
print(type(name))
print(type(baby_age_weeks))
print(type(milk_quantity))
```

## Arithmetic
```python
x = 5
y = 3
print(x + y)
print(x * y)
print(x % y)
```

## Strings
